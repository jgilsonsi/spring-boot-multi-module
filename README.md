# spring-boot-multi-module
Just testing multi-project approach on spring boot

Tests:

* Request a list of users using the following endpoint: http://localhost:8080/users

* Use GET method

* No headers are required

* The output will be:
```json
  [
    {
      "id": 1,
      "name": "Maria"
    },
    {
      "id": 2,
      "name": "Jose"
    }
  ]
```
